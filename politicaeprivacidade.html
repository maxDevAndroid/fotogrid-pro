Política de Privacidade - FotoGridPro

Última atualização: 03/04/2026

1. Introdução

Bem-vindo à Política de Privacidade do aplicativo FotoGridPro, desenvolvido pela empresa TriVisionTech. Esta página explica como coletamos, usamos e protegemos as informações dos usuários do aplicativo.

2. Coleta de Dados

O aplicativo FotoGridPro não coleta dados pessoais identificáveis diretamente. As imagens selecionadas pelo usuário são utilizadas apenas para edição dentro do próprio aplicativo e não são enviadas para servidores externos.

3. Permissões

O aplicativo pode solicitar permissões do dispositivo, como:

- Acesso à galeria/fotos: para permitir que o usuário selecione imagens para edição.

- Armazenamento: para salvar imagens editadas no dispositivo.

Essas permissões são utilizadas exclusivamente para o funcionamento do aplicativo.

4. Publicidade

O aplicativo FotoGridPro pode exibir anúncios fornecidos por parceiros de publicidade. Esses parceiros podem coletar dados anônimos, como identificadores de dispositivo e dados de uso, para personalização de anúncios e análise.

5. Uso de Dados

Os dados coletados por serviços de terceiros são utilizados apenas para melhorar a experiência do usuário e exibir anúncios relevantes. A TriVisionTech não vende, compartilha ou distribui dados pessoais dos usuários.

6. Segurança

Adotamos medidas razoáveis para proteger as informações dos usuários. No entanto, nenhum sistema é totalmente seguro, e não podemos garantir segurança absoluta.

7. Alterações nesta Política

A TriVisionTech pode atualizar esta Política de Privacidade periodicamente. Recomendamos revisar esta página regularmente para se manter informado.
